# Okami Maturity

Landing page and documentation for **Okami Maturity**, the OkamiOps OWASP SAMM/AppSec maturity assessment experience.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Routes:

- `/` — bilingual landing page.
- `/docs/` — OWASP SAMM/AppSec maturity documentation and 90-question assessment structure.
