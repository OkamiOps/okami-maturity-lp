O que é Okami Maturity? What is Okami Maturity?

Okami Maturity usa o modelo OWASP SAMM para avaliações de AppSec: 90 perguntas estruturadas por prática e nível de maturidade 0-3, com evidências e interpretação. Okami Maturity uses the OWASP SAMM model for AppSec assessments: 90 questions structured by practice and maturity level 0-3, with evidence and interpretation.

14security

Cuidados de segurança ao conectar dados reais. Security notes for real data integrations.

  • Não versione assessment data store (local JSON), .env.local, private keys, senhas ou dumps de sessão.Do not version assessment data store (local JSON), .env.local, private keys, passwords or session dumps.
  • Prefira usuário SSH com permissões mínimas para leitura e comandos específicos.Prefer an SSH user with minimal permissions for reading and specific commands.
  • Revise as rotas de import de evidências e comandos permitidos antes de expor a terceiros.Review evidence import routes and allowed commands before exposing to others.
  • Mantenha o backend atrás de token, VPN, Cloudflare Access ou rede privada quando usar dados reais.Keep the backend behind a token, VPN, Cloudflare Access or private network when using real data.
  • Redija logs e documentos antes de renderizar conteúdo em ambientes compartilhados.Redact logs and documents before rendering content in shared environments.

15troubleshooting

Problemas comuns. Common issues.

mock

A UI só mostra dados mock.The UI only shows mock data.

Verifique se o mock SAMM está carregado e se #samm-question-root renderizou as 90 perguntas.Verify that SAMM mock is loaded and that #samm-question-root rendered the 90 questions.

sse

Stream fecha ou não atualiza.Stream closes or does not update.

Teste /stream, confirme proxy sem buffering e reduza temporariamente OKAMI_STREAM_INTERVAL_MS.Test /stream, confirm proxy without buffering and temporarily reduce OKAMI_STREAM_INTERVAL_MS.

import

Falha no import de evidências.Evidence import fails.

Confira o conector de import de evidências e as fontes configuradas (Git, CI artifacts ou upload manual).Confirm the connector for evidence import and configured sources (Git, CI artifacts or manual upload).

samm

Seção de maturidade não renderiza.Maturity section does not render.

Confira se id="samm-question-root" existe no HTML e se renderSAMMQuestions() executou sem erro.Check that id="samm-question-root" exists in HTML and renderSAMMQuestions() ran without error.

write

Arquivo não salva.File does not save.

O import de evidências respeita escopos permitidos. Ajuste o caminho conscientemente.Evidence import respects allowed scopes. Adjust the path intentionally.

build

Build quebra.Build fails.

Rode npm install, confirme Node >= 18 e execute npm run build antes de abrir PR.Run npm install, confirm Node >= 18 and execute npm run build before opening a PR.
